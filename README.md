# POLITICA -ポリティカ-

政界を舞台にした陣取り＆カードのブラウザボードゲーム。政治家を集め、法案を成立させ、
**自分のイデオロギー（思想）のポイントを誰よりも早く貯めた者**が天下を取る。

## ▶ ブラウザで遊ぶ
**https://kuso-von-bazu.github.io/politica-web/**

PC・スマホのブラウザでそのまま遊べます（インストール不要）。

## 遊び方（概要）
- 双六なしのコマンド選択型。自分の手番に **法案を引く / 政治家を入替 / 法案を提出 / 選挙を行う / 買収** を各1回ずつ実行。
- 議員は常に5名。5名の思想値の合計が**影響力**＝採決・選挙の票数。
- 法案は **自分の地盤・信用・場で優勢な思想** だと真価を発揮して追加IP。
- いずれかのイデオロギー（資本／軍国／共産／科学／環境）の**IPを勝利値まで最初に貯めれば勝利**。
- 詳しくは画面右上の「遊び方」、または [遊び方マニュアル.md](遊び方マニュアル.md)。

## 構成
| ファイル | 役割 |
|---|---|
| `index.html` / `style.css` | 画面 |
| `data.js` | カード・盤面・定数（`カード一覧.xlsx`から機械抽出＋バランス調整） |
| `engine.js` | ルールエンジン（非同期choiceプロバイダ方式・Node/ブラウザ両対応） |
| `ai.js` | 対戦AI |
| `ui.js` | 盤面描画・操作 |
| `test_autoplay.js` | Nodeでの自動対戦検証（`node test_autoplay.js 30`） |
| `assets/` | カード・紋章のイラスト（GPT Codexで生成・[画像生成/](画像生成/README.md)参照） |
| `設計メモ_仮決定.md` | ルールの解釈・バランス調整の記録 |

## ローカルで動かす
`起動.bat` をダブルクリック（ローカルHTTPサーバが立ち上がる）。
または `index.html` をブラウザで直接開く。
