# POLITICA デジタル版 設計メモ / 仮決定

原作プロトタイプ（同梱の `POLITICA.docx` / `カード一覧.xlsx` / `マップ等.pptx` / `カード台紙.pdf`）を
参照してデジタル化した。原作は手作りプロトタイプで一部ルールが未確定・矛盾するため、
**デジタル化にあたり以下を仮決定・明確化・簡略化**した。差し替えたい場合は該当箇所を編集すること。

## データ抽出元
- カード4種は `カード一覧.xlsx` から機械抽出（`画像生成/` の生成スクリプト群を参照）。
  - 政治家 49(原作) / チャンス 23 / インシデント 23 / 法案・政策 20。
  - xlsx の「合計」行が政治家として混入していた不具合は除外済み。
  - **無主義**列の影響力を復活（大澤・天和・端本 等。総影響力には数えるが特定イデオロギーIPには寄与しない＝中立の票数要員）。
  - インシデント/法案はイデオロギー変化が「名前行の1つ上の行」に書かれたカードがあり、それを補正して紐付けた。
- 盤面は `マップ等.pptx`／`.pdf` の外周トラック＋中央三角マップを参照。

## バランス調整(2026-06-28)
原作は科学/環境の政治家が3-4枚しかなく構築不能、かつインシデント/法案のデルタが特定思想に偏り、
軍国主義が約50%勝利という偏りがあった。以下で5思想ほぼ均等(各18〜23%・winIP14/20)に調整。
- **科学/環境の政治家を補充**（捩り名スタイルで計17枚追加→各思想pool約50・枚数12-14枚に均衡。計66枚）。
- **インシデントはIPを直接変動させず「世論(climate)」に蓄積**（ゼロサム化して思想偏りを排除）。
  世論が自分の最強イデオロギーを後押ししていると、IPマスで+1ボーナス。
- **法案IPは正のデルタのみ獲得**（法案を通して自IPが減らない）。賛同者はその半分。
- **イデオロギー補正 `IDEO_WEIGHT`**（data.js）でIP獲得に重みを掛け勝率を均一化。
  現値 `{cap:1.0, mil:1.04, com:1.43, sci:1.0, env:0.86}`（端数はキャリーで滑らかに適用）。
  → 再調整は `_tune.js`(任意重み×複数シードで勝率測定)、全体は `_balance.js` で計測。

## 勝利条件
- **いずれかのイデオロギーIPが勝利値（設定で 10／14／20）に到達**した者の勝ち。これ一本に統一。
- **基幹政策は「即勝利」を廃止**（2026-06-28・ユーザー決定）。1票でゲームが終わる上に勝率が
  振れて調整困難だったため。代わりに下記「強力な永続政策」とした。

## イデオロギーとIP
- 5イデオロギー: 資本主義 / 軍国主義 / 共産主義 / 科学主義 / 環境主義。
  （原作マップの「社会主義」は共産主義と同一視）。
- **影響力** = スロットの政治家カードのイデオロギー値の合計。投票・採決の票数に使う。
- **一番影響力の高いイデオロギー** = IPマス取得・基幹政策の対象になる。
- 政治家の「無主義」値はデジタルでは未使用（影響力に算入せず）。

## 盤面（外周24マス＝7×7グリッドの外周セル）
- 総選挙 / 政治家獲得 / IP / チャンス / インシデント / 法案提出 / 献金(G+2) / 一回休み を配置。
- サイコロ1個で時計回りに進む。
- **総選挙**: 首班がこのマスを通過/到達すると、その手番終了時に首班指名選挙。

## マス効果
- **政治家獲得**: 2枚引いて1枚をスロット（最大5）へ。1G or 1IP 支払いで3枚から選択。
  「赤軍の設立」成立中は首班のみ3枚。
- **IP**: 一番影響力の高いイデオロギーのIPを+1。
- **チャンス**: 1G or 1IP 消費で1枚引く（手札上限3、「独占禁止法」成立中は1）。
- **インシデント**: 強制1枚。引く前に 2G or 2IP で裏のまま回避可。
  効果＝イデオロギー変化を **全員のIPに適用**（0未満は0）＋カード固有効果。
- **法案提出**: 法案を1枚引き、続けて採決にかけられる。
- **献金**: +2G。 **一回休み**: 次の手番を飛ばす。

## 法案・政策
- **必要影響力** = カードのイデオロギー変化量の絶対値合計 × 2（最低3）。原作に数値が無いため算出式を仮決定。
- **採決**: 提出者は自動賛成。他プレイヤーが賛成/反対。賛成側影響力 > 反対側影響力 かつ
  賛成側影響力 ≥ 必要影響力 で可決。
- **可決時**: 提出者は **正のイデオロギー変化のみ** をIPとして獲得（IDEO_WEIGHT補正・端数キャリー）＋信用度変化(pip)。賛同者はその半分。
- 成立法案は場に最大5枚（恒常効果カードのフラグを適用）。6枚目で最古を廃案。
- 「行政カード」「廃案2/3」「マニフェスト」は簡略化（マニフェストは首班特典の信用度+5に一般化）。

### 基幹政策(強力な永続政策)
- 影響力12以上で、自分の最強イデオロギーの基幹政策を提出できる（1人1枚まで）。
- 可決すると、その思想IPを **+5(固定・補正なし)** 獲得し、**廃案不可の永続政策**として基幹政策スロットへ。
- 即勝利ではなく、終盤の強力な決め手。対戦AIは脅威となる提出者の基幹政策には反対する。

## チャンスカード
- **信用度**（初期10）を `|信用度コスト|` 消費して使用。信用度が足りないと使えない。
- 実装済み効果: 不信任決議/襲撃/行財政改革/スキャンダル/包囲網/政治献金/名演説/機関紙発行/
  一日天下/暗殺/財政の崖/世論操作/均衡/決闘/集中審議/讒言/陳情/ねずみとり/恩赦/転向。
- 静観・連合・牛歩戦術はテキスト表示のみ／効果限定（ターン継続型の複雑効果は簡略）。

## 首班指名選挙
- 各プレイヤーが立候補可否を選択 → 非立候補者は立候補者の誰かへ投票（票＝影響力）。
- 最多得票者が首班（首相）。「文民統制」成立中は軍国主義筆頭は首相になれず次点へ。
- 当選時、党首が組閣 → 信用度+5（原作の政治家別ボーナスを一般化）。
- 「現首相続投」「マニフェスト無条件成立」などは簡略化。

## 買収
- 2G or 2IP 消費で他プレイヤーの政治家1枚を自分のスロットへ（自分が5枚未満のとき）。
  原作のイデオロギー一致/不一致による2IP/5IPの差は簡略化（一律2）。

## 政治家の特殊能力
- カードのテキストは保持・表示するが、効果実装は **「党首就任で信用度+5」系の一般化**のみ。
  個別の複雑効果（毛沢南・加流部人良 等）はテキスト参照に留める。

## AI
- ヒューリスティック: 自分の最強イデオロギーを伸ばす法案/カードを優先、脅威プレイヤーを妨害、
  影響力上位なら立候補。完全情報・非交渉。

## 既知の簡略化（今後の拡張余地）
- 交渉/共同提出の能動的な相談、行政カードスロット、マニフェスト掲示、政治家個別能力のフル実装。
