# POLITICA -ポリティカ- 遊び方マニュアル

政界を舞台にした陣取り＆カードゲーム。政治家を集め、法案を成立させ、
**自分のイデオロギー（思想）のポイントを誰よりも早く貯めた者**が天下を取る。

## 起動
- `起動.bat` をダブルクリック（ローカルサーバが立ち上がりブラウザで開く）。
- うまくいかない場合は `index.html` をブラウザに直接ドロップしてもOK。

## 目的（勝利条件）
**いずれかのイデオロギーのIP（イデオロギーポイント）を勝利値まで最初に貯めた**プレイヤーの勝ち
（ゲーム設定で 10／14／20 を選択。標準14）。

## 5つのイデオロギー
資本主義 / 軍国主義 / 共産主義 / 科学主義 / 環境主義。
プレイヤーパネルの5色チップが各IP。白枠の付いた色が「あなたの一番影響力の強い思想」。

## 基本の流れ
1. **サイコロ前の行動**（任意）: チャンスカード使用・法案提出・買収。
2. **サイコロを振る**: 出目の数だけ盤面を時計回りに進む。
3. **止まったマスの効果**を解決。
4. **追加行動**（任意）。
5. 手番終了。首班が「総選挙」を通過していたら**首班指名選挙**。

## マスの説明
| マス | 効果 |
|---|---|
| 🗳 総選挙 | 首班が通過すると首班指名選挙が発生 |
| 👤 政治家獲得 | 山札から2枚引き1枚をスロット（最大5）へ。1G/1IPで3枚から選択 |
| ★ IP | 一番影響力の高い思想のIPが+1 |
| 🎴 チャンス | 1G or 1IPでチャンスカードを引く（手札3枚まで）|
| ⚡ インシデント | 強制発生。世論（思想の潮流）が変動し、追い風の思想はIPマスでボーナス。2G/2IPで回避可 |
| 📜 法案提出 | 法案を引き、採決にかけられる |
| 💰 献金 | +2G |
| 💤 一回休み | 次の手番を飛ばす |

## 政治家と影響力
- スロットの政治家カードの思想値の合計が、あなたの**影響力**。
- 影響力は採決・選挙の「票数」になり、強い思想がIP獲得・基幹政策の対象になる。
- **無主義**の政治家は影響力（票数）にはなるが、特定思想のIPは伸ばさない中立要員。
- 党首＝最も影響力の高い政治家。首班になると党首が首相として組閣する。

## 法案の採決
- 提出には**必要影響力**が要る（賛同者の影響力で補える）。
- 賛成側の影響力が反対側を上回り、かつ必要影響力に達すれば**可決**。
- 可決すると提出者は思想を伸ばす分（正の変化）のIPと信用度を獲得（賛同者は半分）。
- 成立法案は場に最大5枚。恒常効果を持つものもある。

## 基幹政策（強力な永続政策）
- 影響力12以上で、自分の最強思想の**基幹政策**を提出できる（1人1枚）。
- 可決すると、その思想IPを一気に **+5** 獲得し、**廃案されない永続政策**になる。
- 即勝利ではないが、終盤で勝ちを手繰り寄せる切り札。相手は脅威なら反対してくる。

## チャンスカード
- 使用には**信用度**を消費（初期10）。強力なカードほど信用度を多く食う。
- 妨害（襲撃・スキャンダル・暗殺）、強化（政治献金・世論操作）など系統さまざま。

## 首班指名選挙
- 立候補するか選び、非立候補者は誰かに投票（票＝影響力）。
- 最多得票者が首班（首相）。首班には組閣ボーナス（信用度+5）。

## 操作のコツ
- 画面下の**決定バー**に選択肢が出る。あなたの手番だけ操作可能。
- 右側パネルで全員のIP・資金・影響力・政治家を確認できる。
- まず政治家を集めて影響力を高め、自分の思想を伸ばす法案を通すのが王道。

詳しいルールの解釈・簡略化点は `設計メモ_仮決定.md` を参照。
